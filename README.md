# Online Exam System

This archive is split into:

- `frontend/` → deploy to Vercel
- `backend/` → deploy to Render

## Frontend
1. Deploy `frontend/` to Vercel.
2. Open `frontend/config.js`.
3. Replace the backend URL placeholder with your Render URL.

## Backend
1. Deploy `backend/` to Render.
2. Set `MONGODB_URI` in Render environment variables.
3. Use `npm install` as build command and `npm start` as start command.
